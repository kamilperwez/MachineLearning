## 📖 Overview

This repository contains the data exploration, model training, and evaluation scripts for several distinct machine learning problems. The goal of this project is to demonstrate end-to-end ML workflows—from raw data preprocessing to building an interactive deployment interface.

For each dataset, 5 to 8 different machine learning models were trained and compared to find the most accurate predictive engine.

## 🗄️ Included Projects & Datasets

1. **Heart Disease Prediction (Classification)**
   * Predicts the likelihood of heart disease based on patient health metrics.
2. **Predicting Crop Yields (Regression)**
   * Forecasts agricultural crop yield outcomes based on environmental and regional inputs.
3. **Laptop Price Predictor (Regression)**
   * Estimates the market value of laptops based on hardware specifications.
4. **Car Price Predictor (Regression)**
   * Evaluates the resale value of vehicles based on historical data and features.

## 🛠️ Tech Stack & Libraries

* **Language:** Python 3.x
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Web Framework / Deployment:** Streamlit / Vercel (via the interactive dashboard)

## 🧠 Methodology

1.  **Data Preprocessing:** Handled missing values, encoded categorical variables (One-Hot Encoding/Label Encoding), and scaled numerical features.
2.  **Exploratory Data Analysis (EDA):** Analyzed feature distributions, correlations, and target variable balance to understand the underlying data structures.
3.  **Model Selection & Training:** Evaluated multiple algorithms per project, including:
    * Logistic Regression / Linear Regression
    * Random Forest (Classifier & Regressor)
    * Support Vector Machines (SVC & SVR)
    * Gradient Boosting (XGBoost)
    * Decision Trees
4.  **Evaluation:** Models were assessed and ranked using Accuracy, Precision, Recall, and F1-Score for classification, and RMSE/R-squared for regression tasks.
